# Minimal Tide Fast Path Checklist

1. Create `v1` as wave height input.
2. Create `v2` as wave result output.
3. Create the first `fun` and write the sine-wave text logic.
4. Switch to `interventionerScene`.
5. Bring `v2` into the intervention view.
6. Create the second `fun` and write the display shader text.
7. Press `Build`.
8. Inspect `algorithmLib/algorithmSrc/minimal_tide_fastpath/`.
9. Inspect `algorithmLib/algorithmruntimeLib/minimal_tide_fastpath/`.
10. Open `debugTool` and look for `minimal_tide_fastpath`.
